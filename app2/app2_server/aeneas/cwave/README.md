# aeneas.cwave

**aeneas.cwave** is a Python C extension to read WAVE files.

## API

See the [__init__.py](__init__.py) file.

## Compiling the Python C extension locally

```bash
$ make py 
```

## Compiling the pure C driver program

```bash
$ make driver
```

## Testing the pure C driver program

```bash
$ make test
```
